# **$ Linux**

- ctrl + l -> clear
- ↑で過去のコマンドを選択できる
- mkdir -> make directory
- cat -> ファイルの中身を表示
- ls -> (list segments) ファイルやフォルダの一覧を表示
- -> ls -la: -l -> longの意。詳細。  -a -> allの意。全て。
- ll -> ls -l のエイリアス。
- rm -> ファイルの削除
- rm -r -> ディレクトリの削除
- mv -> ファイルの移動
- grep -> unixのコマンド。指定した文字列を検索する。
- -v -> 一致しないもの
- | -> でコマンドをつなげて指定することができる

## vim

- i -> insertモードに。
- :wq -> 保存して終了
- :q! -> 保存せずに終了
- / -> 検索。この状態でescapeして n -> 次へ N -> 前へ
